算法图解01：算法运行时间

发表于 2019-08-10 | 分类于 算法图解

1. 首先要明确算法与代码之间的关系：

  • 算法即计算方法，是用于解决某类问题的指导思想，这是一个抽象的概念。
  • 代码则是解决某个具体问题的具体措施，它可以认为是算法的具体表现，这是一个具体的概念。
  • 算法与代码的关系可以认为是Python中类对象与实例对象的关系。

2. 在明确算法与代码的关系后，则要明确算法运行时间与代码运行时间的关系：

  • 代码运行时间是一个具体的概念。即确定操作数后，执行该代码所需要花费的时间。
  • 算法运行时间是一个抽象的概念。即随着操作数的增加，代码运行时间的增加速度。
  • 注释：操作数指的是代码所需要处理的元素数量。

3. 在确定算法运行时间的概念后，则要对算法运行时间这一抽象概念进行描述。这里我们一般采用大O表示法

大O表示法描述的是：最不利情况下，代码运行时间与操作数之间的函数类型

概念解释如下：

  • 最不利情况是指：
  • 函数类型指的是：我们不关心“代码运行时间-操作数”函数的具体表达式，只关心该函数的函数类型，如直线型、指数型、对数型等。

4. 在明确大O表示法后，则要了解一些常见的大O运行时间

  • O( log n )，即对数时间。代表算法——“二分查找”
  • O( n )，即线性时间。代表算法——“简单查找”
  • O( n * log n )
  • O( n2 )
  • O( n! )
  • O( 1 ),即常量时间
  • 注意：在算法内容中log n 均表示为log2 n
