正则表达式(Regular expression)

正则表达式又叫做规律表达式，通过给特定的符号赋予其通配的功能，从而使一个简短的内容能够表达成符合多种情况的内容。

  • 即正则表达式是一个抽象的定义，它有好多种具体化的内容。
  • 就如同苹果可以指代红苹果和绿苹果

一、基本语法

　 元字符 作用 示例 备注
重复 {m,n} 可匹配重复前一个字符m次至n次 ca{1,3}t可匹配cat、caat与caaat {，1}为0-1次，{1，}为1-无穷次
* 可匹配重复前一个字符零次或多次，相当于{0,} ca*t 可匹配ct、cat、caat… 　
+ 可匹配重复前一个字符一次或多次，相当于{1,} ca+t可匹配cat、caat…，但不能匹配ct 　
？ 可匹配重复前一个字符零次或一次，相当于{0,1} ca?t可匹配ct与cat 　
指代 . 可匹配除换行符外其他任何字符 .at可匹配aat、bat、cat… 　
[   ] 用于构造可匹配的指代字符 [ac]at可匹配aat、cat，但不能匹配bat 可用“-”来选择范围
如[1-3]表示[123]；[a-c]表示[abc]
但注意[1-255]表示[1255]
^ 可匹配出除后一字符外的其他任一字符 [^3]可匹配除3外的任一字符，如0,1,2 只能至于中括号中第一个位置，否则表示^本身
或者 | 可匹配前一个字符或后一个字符 12|3只能匹配12或13 | 的优先级非常低。例如 Fish|C 应该匹配 Fish 或 C
打包 (   ) 将多个字符打包成一个子组，可用于重复、或者，且可用于查询 ([1-2][3-4]){1,2}可匹配1314,2424… 　
反斜杠\ 钝化其后的元字符的作用，使之只能匹配它本身 \*只能匹配*,\^只能匹配^ 　
\d 匹配任一十进制数字；相当于[0-9] \d\d可匹配18 　
\D 与 \d 相反，匹配任何非十进制数字的字符；相当于类 [^0-9] \d\D可匹配1a，但不能匹配18 　
\s 匹配任何空白字符（包含空格、换行符、制表符等）；相当于类 [ \t\n\r\f\v] 　 　
\S 与 \s 相反，匹配任何非空白字符；相当于类 [^ \t\n\r\f\v] 　 　
\w 匹配任何单词字符 \w\w可匹配ba、bs、cw… 　
\W 于 \w 相反 \W可匹配’.'、’!'等非单词字符 可用于查找单词间的空格，以分割各单词等
\b 匹配单词的开始或结束 \bword\b，在'my word'中有匹配对象
\bword\b，在‘myword’中无匹配对象		 可用于查找具体单词，而不会匹配到相似的内容
\B 与 \b 相反 　 　
首末 ^ 匹配字符串的起始位置 ^my 在‘my word'中有匹配对象
^my 在’word my‘中无匹配对象		 如果设置了 MULTILINE 标志，就会变成匹配每一行的起始位置
$ 匹配字符串的结束位置，每当遇到换行符也会离开进行匹配 my$ 在‘my word'中无匹配对象
my$ 在’word my‘中有匹配对象		 　

正则表达式中反斜杠“\”具有其特殊用途，而Python中“\”又有表示转义的作用。故正则表达式必须使用原始字符串r’string’，先钝化Python中”"的转义功能，才能保证正则表达式中“\”特殊功能的正常使用。

二、模块对象

re.compile(pattern[,flags=0])
返回的是一个模块对象（re.Pattern object），作用是编译正则表达式并将其存储起来方便之后调用。
再次强调，对于使用了反斜杆“\”的正则表达式，必须使用原始字符串来钝化Python自身对反斜杆的转义功能。。

模块对象（re.Pattern object）拥有以下方法：

方法 概念
mathch( string ) 只有从开始处符合正则表达式匹配，才算作匹配
search( string ) 遍历字符串，找到正则表达式匹配的第一个位置
findall( string ) 遍历字符串，找到正则表达式匹配的所有位置，并以列表的形式返回
finditer( string ) 遍历字符串，找到正则表达式匹配的所有位置，并以迭代器的形式返回

如果没有找到任何匹配的话，match() 和 search() 会返回 None；如果匹配成功，则会返回一个匹配对象（match object），包含所有匹配的信息：例如从哪儿开始，到哪儿结束，匹配的子字符串等等。

匹配对象（match object）具有以下方法：

方法 功能
group() 返回匹配的字符串
start() 返回匹配的开始位置
end() 返回匹配的结束位置
span() 返回一个元组表示匹配位置（开始，结束）

三、模块级别的函数

若正则表达式仅使用一到两次，则不一定要将正则表达式存储于模块对象中，而可以直接调用模块的内置函数。

内置函数有例如 match()，search()，findall()，sub() 等等。

这些函数的第一个参数是正则表达式字符串，其他参数跟模式对象同名的方法采用一样的参数；返回值也一样，同样是返回 None 或者匹配对象。在此不再赘述。
