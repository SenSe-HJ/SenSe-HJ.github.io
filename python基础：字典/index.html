python基础：字典

发表于 2019-08-01 | 分类于 python基础

一、字典的理解

与列表进行对比：

１.列表的每一个索引（Index）都有其对应的值（Value）；

２.字典的每一个键（Key）也都有其对应的值（Value），且键的内容可以由我们编写。

这样，字典的存在方便了我们根据关键字提取对应值了。

二、 字典的创建

2.1.方法一：{Key：Value}  

  • 注意1：是 “：” 而不是 “=”
  • 注意2：是 { } 而不是 [ ]

2.2.方法二：dict（）

该方法有两组表达形式

2.2.1 给键赋值的方法

  • 注意1：若键为字符串，不能加引号“”。否则会报错
  • 注意2：连接键与值的是“=”而不是 “：” ，不要与一般定义混淆。
  • 注意3：该方法中，键不能是数字开头(类似变量名）

2.2.2 将键与值元组打包

  • 注意1：该方法必须创建两对及以上的（key，value），不然或报错。
  • 注意2：多对（key，value）可用列表或元组打包都是可用的。

2.3.内置函数fromkeys（keys，value）

要明确该函数的意义：fromkeys可以批量生成多对“”相同值的而键不同”的“键值”。

  • 注意1：该函数是新建字典，而不改变原字典。
  • 注意2：keys可以是用元组或列表打包，而value只能是一个值。

    2.4.其他注意事项

  • 各对“键-值”在字典中的顺序是无序的，而不是按照添加先后排顺序的。

    三、字典的访问

    3.1.基本访问方法：字典名 [键]

  • 注意：键不存在于字典中时，会报错。

3.2.内置函数get（key）

使用该函数时，键存在于字典中时，会返回对应值；键不存在于字典中时，会返回“None”

3.3.内置函数setdefault（key）OR setdefault（key，value）

使用该函数时，键存在于字典中时，会返回对应值；键不存在于字典中时，可在其后添加对于的值，该函数会将其添加至字典中”

四、字典的修改

4.1.基本修改方法：基本访问+重新赋值

这一块与列表、元组相同，不再赘述。
值得注意的是，若访问的键不存在且对其进行赋值，则系统会自动向字典中添加该对键-值。

  • *get（）与setdefault（）的访问方法不能用于字典的修改 *

4.2.内置函数update（{key，value}）

在使用该函数时，若键存在于字典中，则将相应键值进行修改；若键不存在于字典中，则将其“键-值”添加至字典中。

  • update 与 setdefault 的区别：对于已存在于字典中的键，setdefault将不会去改变它对应的值，而update则会去改变它的值。

五、字典的删除

5.1.外部函数del

使用方法与列表类似，有删除整个字典的存在于删除字典某对“键-值”两个作用

5.2.内置函数pop（key）

该函数可使字典删除指定键的值，并返回该值。（老是忘记）

5.3.内置函数clear（）

清空字典内容
与直接将字典赋值为空字典的区别：

5.3.1 clear方法

结果：a、b均成为了空的字典

5.3.2 直接赋空值法

结果：a成为了空字典，而b还是原来的有值字典。

分析：

(1).在python中，值相当于物体，而变量名相当于标签。
(2).给变量名赋值，就相当于将标签贴于物体之上。而一个物体可以有多个标签，及一个值可以赋予多个变量名。
(3).若将变量名重新赋值，则是将标签贴于其他物体上，而先前的物体及其他的标签不会发生变化。

就此处而言：

(1).“b = a”使a、b标签都贴在了字典{‘one’: 1, ‘two’: 2, ‘three’: 3}上。
(2).而clear函数是将{‘one’: 1, ‘two’: 2, ‘three’: 3}直接变为了{ }，使得贴在其上的a、b值都变成了{ }。(3).而“a={ }”使a 标签重新贴在了字典{ }上，而b标签还是贴在原有的字典上。故a成为了空字典，而b还是原来的有值字典。

六、字典的复制

6.1 内置函数copy（）

该函数属于浅拷贝

浅拷贝与深拷贝的区别（依旧采用标签与物体的解释方法）

  • 浅拷贝只是多给物体贴了一个标签。
  • 深拷贝是指将增加了一个新的物体。

七、遍历字典

7.1.遍历字典的键与值：items（）

  • 注意1：应指定两个变量与函数items（）对应
  • 注意2：遍历的顺序不一定与字典的存储顺序一样

7.2.遍历字典的键：keys（）

6.3.遍历字典的值：values（）

  • 注意：直接使用values函数，系统会遍历所有值（包括重复出现的值）

若想相同的值只访问一次，可使用外部函数set（）（将其转化为集合，删除重复出现的内容）
